## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Anastasios97/Academic-Website-Anastasios-Papadam.git
   cd Academic-Website-Anastasios-Papadam
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

This generates an optimized production build in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

## 🚢 Deployment

### Current Deployment: Vercel

The project is currently deployed on **Vercel** and automatically deploys on every push to the `main` branch.

**Deploy Status**: [![Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?logo=vercel)](https://academic-website-anastasios-papadam.vercel.app)

#### Steps to Deploy on Vercel:
1. Push code to GitHub
2. Connect your GitHub repository to [Vercel](https://vercel.com)
3. Vercel automatically detects Vite and deploys on every push

### Alternative: GitHub Pages

To deploy on GitHub Pages instead:
1. Update `vite.config.ts` to set the base path
2. Enable GitHub Pages in repository settings
3. Use GitHub Actions for automated deployment
